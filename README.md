
Moderated multi-agent group chat.


## Dev Notes

Initialize with `uv`:
- `uv venv`
- `source .venv/bin/activate` # or similar
- `uv sync`

Set configuration:
- copy `.env.template` to `.env`
- edit the `.env`
- copy `neo4j.example.json` to `neo4j.json`
- edit the named Neo4j connections

Run the agent: `adk web`
